# LaMovie SkyStream Plugin

Stream movies from [La.Movie](https://la.movie) in SkyStream.

## Installation

1. Open SkyStream → Settings → Extensions
2. Tap "Add Repository"
3. Enter: `https://raw.githubusercontent.com/rheno911/skystream-lamovie/main/repo.json`
4. Enable **LaMovie** plugin

## Features
- Movies in Spanish (HD quality)
- HLS streams (Vimeo-powered)
- Search & Home sections
- Auto quality selection
